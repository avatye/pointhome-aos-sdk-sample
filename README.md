# pointhome-aos-sdk-sample

안드로이드에서 **포인트홈(PointHome)** 을 연동하기 위한 **코틀린** 예제 코드입니다.  

---

## 프로젝트 구조

- **App.class**  
  포인트홈 SDK초기화를 진행합니다.

- **MainActivity.class**  
  포인트홈 레이아웃 초기화를 진행합니다.

- **Additional.class**  
  포인트홈 추가 API 정의입니다.
